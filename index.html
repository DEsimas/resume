Разработка бота для дискорда на фрилансе

Был реалезован модерационный бот для сервера группы стримеров по техническому заданию (https://www.figma.com/file/f9EEUsgsIHfVp184Ch9azk/Mod-and-Report?node-id=0%3A1&t=qC9wcumD2gsXAk8D-1). Бот обладает функционалом предупреждения, мута, бана пользователей. Позволяет владельцу сервера назначать модераторов и администраторов с различными привелегиями, вознаграждает модераторов за работо внутрисерверной валютой, интегрированной в систему остальных ботов сервера. Для реализации использовались TypeScript, Didcord.js и MongoDB.
