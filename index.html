Разработка модерационного бота для Discord

Успешно выполнил заказ на фрилансе. Нужно было создать модерационного бота для платформы Discord на TypeScript с использованием MongoDB. Его функционал включает выдачу различных наказаний на определённое время, сохранение их истории, внутрисерверную валюту для модераторов, систему жалоб, интеграцию с другими ботами.
Техническое задание: https://www.figma.com/file/f9EEUsgsIHfVp184Ch9azk/Mod-and-Report?node-id=0-1
